# Automação de Planilhas Excel com Python

Este projeto automatiza o processamento de dados de vendas, calculando impostos e gerando relatórios formatados.

## 🛠️ Tecnologias
- Python 3.x
- Pandas
- Openpyxl

## 🚀 Como executar
1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute o script: `python main.py`
